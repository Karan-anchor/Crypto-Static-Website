# 🚀 Crypto Static Website

A modern and responsive cryptocurrency landing page built with **React.js**, **Vite**, and **Tailwind CSS**.

## ✨ Features

- Responsive Design
- Modern UI
- Hero Section
- FAQ Section
- Contact Form
- Footer with Social Links
- Smooth Animations

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Icons

## 🚀 Installation

```bash
git clone https://github.com/Karan-anchor/Crypto-Static-Website.git
cd Crypto-Static-Website
npm install
npm run dev
```
